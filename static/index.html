About this website

I'm Matteo Silvestro and I'm learning Japanese. Lately I've been enjoying a lot Japanese songs, especially anisong and J-pop. I've been wanting to understand the lyrics for so long, but it's never easy to find good translations (Japanese is hard to translate).

So I thought: why not create a website when I can share my attempts at lyrics translation? This way I can practice and people might find it useful to better understand their favorite songs. They can even contribute, if they wish to.

The name is an abbreviation of リリック (ririkku), which loosely means "lyrics", and コンメンタール (konmentaaru), which means "annotation". Many Japanese abbreviations take the first two syllables of each word (examples: パソコン, ポケモン, ...) to form a new one, so in this case the result is リリコン (ririkon).

For this type of annotated translation my source of inspiration has been Satori reader. I highly recommend it to whoever wants to learn Japanese. It has been great in improving my understanding of the language and is more than worth its price. Since I found its concept so useful and ingenious, I wanted to let everyone (including me) easily create websites with similar annotated translations. For this purpose, I created a specific markup for Emerald files that can be processed by Emeraldya to output HTML files. You can use it as well, as I do for this website.

The source of dictionary definitions is Jisho.
